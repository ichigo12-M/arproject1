# AR Notice Board App

## 📌 Overview

This project demonstrates a camera-based augmented reality (AR) simulation built using Unity.
The application uses the device camera and displays a notice board dynamically when the user interacts with the interface.

---

## 🎯 Features

* Live camera feed integration
* Interactive UI button
* Dynamic notice board display
* Lightweight and works on most Android devices

---

## 🛠️ Technologies Used

* Unity Engine
* C# Programming
* Unity UI System
* Mobile Camera Integration

---

## 🚀 How It Works

1. The app opens the device camera
2. A button is displayed on the screen
3. When the button is pressed, a notice board appears
4. The notice board can be used to display announcements or information

---

## ⚠️ Note

This project uses a camera-based AR simulation instead of full ARCore implementation due to device compatibility limitations.

---

## 📱 How to Run

1. Build APK from Unity
2. Install on Android device
3. Allow camera permission
4. Tap the button to view the notice

---

## 📂 Project Structure

* Assets/ → Main project files
* Scripts/ → C# scripts
* UI/ → User interface elements
